Detailed Description

Contains structure and methods for spinal cord mesh.

This class represents spinal cord mesh, heriting from Mesh. A spinal cord mesh is a triangular mesh with a specific structure : stack of disks. Each disks have the same number of points and triangles are formed between two adjacent disks.

Methods for saving mesh, centerline and computing cross-sectional areas are available.
