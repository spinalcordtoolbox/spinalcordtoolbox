Detailed Description

Compute the orientation of the spinal cord using circular Hough transform and apply the rotation around a point on the mesh.

This class compute the orientation of the spinal cord by extracting a 2D image (5cm X 5cm) in the orientation of the last iteration of segmentation mesh and performing a circular Hough transform on it. The center of the first detected circle is assume to be the center of the spinal cord.

As an error metric, the distance between the detected position and the center of the 2D image is computed and return as reference. If this distance is larger than 3 cm, the return value is false, true if smaller. The transformation (rotation around the center of mass of the first disk of the tubular mesh) is apply on the mesh if the angle of rotation is between 1 and 3 degrees. Small values don't change anything valuable on the mesh orientation and large values are assumed to be false.
