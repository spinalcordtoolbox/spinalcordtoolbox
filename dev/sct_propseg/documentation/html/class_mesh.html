Detailed Description

Contains structure and methods for triangular mesh.

A triangular mesh is represented by a set of points (vertices) and a connectivity table (triangles). Multiple methods are available as computation of triangles center of mass, transformations on the mesh, saving, decimation and smoothing, computation of distance between meshes, etc.
