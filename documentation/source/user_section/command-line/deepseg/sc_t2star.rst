sc_t2star
=================
                
This model has been retired. Please switch to the contrast-agnostic spinal cord model (``sct_deepseg spinalcord``) for further segmentation of T2*-weighted spinal cords. If you are segmenting an EPI-BOLD fMRI sequence, consider using ``sct_deepseg sc_epi`` instead.
