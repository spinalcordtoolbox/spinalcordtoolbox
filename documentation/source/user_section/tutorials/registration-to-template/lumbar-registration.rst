.. _lumbar-registration:

Registering lumbar images to the PAM50 template
###############################################

This tutorial demonstrates how to `register <http://jpeelle.net/mri/image_processing/registration.html>`_ a lumbar anatomical MRI scan to the PAM50 template. T2w images are used for this tutorial, due to lumbar segmentation model only being available for the T2w contrast.
